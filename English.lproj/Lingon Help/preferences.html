External editor:

The application that should open the selected plist when choosing "Open In External Editor" in the File-menu. Note that Lingon does not automatically update if changes are made to the plist in an external editor; one needs to perform a Refresh in Lingon after it is saved in the external application.

Text font:

The font which the text in Expert-view and the man pages are displayed

Table font size:

Choose between 11 and 13 points as the font size for the tables

Check for updates at startup:

If you are connected to the internet it will check if there's a newer version of Lingon when it starts up, and if so, present you with a choice to download it

Warn me:

If you feel like you want to scream every time the warning about modifying the Apple-supplied configuration files comes up, uncheck this box
