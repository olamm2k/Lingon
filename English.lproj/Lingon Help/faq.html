What is it?

Lingon lets you control, edit and create configuration files for launchd

Requirements?

Mac OS X Tiger 10.4.2 or later

The name?

Lingon is the Swedish name for lingonberry and hence the icon

Why have you written it?

It has been more complicated than it should have to be to create and control the configuration files for launchd and my humble hope is that now people can discover how great launchd really is for Mac OS X

How do I uninstall it?

Drag Lingon to the trash, and you can find the preference file at /Users/<your home folder>/Library/Preferences/org.lingon.Lingon.plist. And there is a folder called Lingon in /Users/<your home folder>/Library/Application Support if you've ever authenticated
